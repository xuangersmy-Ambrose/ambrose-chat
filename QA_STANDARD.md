# AMBROSE 质量保证标准 (QA Standard)

## 一、自检清单（每次提交前必须完成）

### 代码层面
- [ ] 所有函数有错误处理 (try-catch)
- [ ] 用户输入有验证
- [ ] 无重复函数定义
- [ ] 动态DOM有清理机制
- [ ] 无console.log调试代码

### 功能层面
- [ ] 页面切换正常
- [ ] 数据保存成功
- [ ] 无控制台报错
- [ ] 响应式布局正常
- [ ] 移动端兼容

### 性能层面
- [ ] 页面加载 < 3秒
- [ ] 内存占用稳定
- [ ] 无内存泄漏

---

## 二、错误处理规范

### 必须使用的模式
```javascript
// ✅ 正确：有错误处理
function doSomething() {
  try {
    // 操作
  } catch (e) {
    console.error('错误:', e);
    showToast('操作失败，请重试');
  }
}

// ❌ 错误：无错误处理
function doSomething() {
  // 操作
}
```
