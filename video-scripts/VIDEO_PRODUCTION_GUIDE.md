## 快速制作流程（每视频5分钟）

### 步骤1: 下载素材（1分钟）
```
1. 打开 Pexels 搜索 "push up" 或 "squat"
2. 下载2-3个相关视频片段
3. 保存到电脑文件夹
```

### 步骤2: 剪映图文成片（2分钟）
```
1. 打开剪映 → 点击"图文成片"
2. 选择"图文成片"功能
3. 复制下方文案粘贴进去
4. 选择"健身"类模板
5. 点击生成
```

### 步骤3: 替换素材（1分钟）
```
1. 删除自动生成的不相关素材
2. 导入下载的健身视频片段
3. 按脚本时间轴排列
```

### 步骤4: AI配音（1分钟）
```
1. 点击"文本朗读"
2. 选择"解说小帅"或"解说小美"
3. 调整语速为1.2倍
4. 生成配音
```
