# AMBROSE Health v1.2 - 内容库与课程系统

## 🎯 功能规划

### 1. 运动视频课程库
- **居家健身** (无器械) - 15-30分钟短视频
- **办公室拉伸** - 5-10分钟快速放松
- **瑜伽入门** - 从基础到进阶
- **HIIT训练** - 高强度间歇训练
- **有氧运动** - 燃脂/心肺提升

### 2. 健康食谱库
- **减脂餐** - 低卡高蛋白
- **增肌餐** - 高蛋白碳水配比
- **快手早餐** - 5-10分钟搞定
- **营养配搭** - 均衡营养指南

### 3. 养生知识库
- **中医养生** - 体质调理
- **节气调理** - 四季养生
- **睡眠改善** - 科学助眠
- **压力管理** - 冥想放松

### 4. 收藏与学习
- 收藏课程/食谱/文章
- 学习计划制定
- 学习进度追踪

---

## 🛠 技术方案

### 内容存储
- 使用JSON文件存储课程数据（初期）
- 视频使用外部链接（B站/YouTube嵌入）
- 图片使用CDN或Base64

### UI设计
- 课程卡片网格展示
- 分类Tab切换
- 详情页播放/阅读
- 收藏按钮

### 数据结构
```javascript
// 课程
course: {
  id, title, category, difficulty,
  duration, calories, image, videoUrl,
  description, steps, tags, isPremium
}

// 食谱
recipe: {
  id, title, category, calories,
  time, image, ingredients, steps,
  nutrition, tags, isPremium
}

// 文章
article: {
  id, title, category, readTime,
  image, content, tags, isPremium
}
```
